Publicar Grátis no GitHub Pages

O GitHub Pages permite que você coloque este aplicativo do Clave na internet totalmente de graça e sem precisar programar nada extra. Siga este passo a passo:

1

Baixe este arquivo HTML

Baixe este código gerado e renomeie o arquivo no seu computador exatamente para: index.html

2

Crie uma conta no GitHub

Acesse github.com, faça seu cadastro (é gratuito!) e clique no botão verde New Repository (Novo Repositório).

3

Configure o Repositório

Dê um nome para ele (ex: minhas-partituras), deixe-o como Public, selecione a opção "Add a README file" e clique em "Create repository".

4

Suba o seu arquivo index.html

Dentro da página do seu repositório, clique em Add file > Upload files. Arraste e solte o seu arquivo index.html ali e depois clique no botão verde Commit changes no final da página.

5

Ative o GitHub Pages

Na página do repositório, clique na aba superior Settings (Configurações). No menu esquerdo, vá em Pages. Na seção "Build and deployment", mude o Branch de "None" para main (ou master) e clique em Save.

Pronto!

Em menos de 2 minutos, o GitHub irá gerar um link público (como https://seu-usuario.github.io/minhas-partituras) que você poderá abrir em qualquer computador, celular ou tablet para gerenciar suas músicas de onde estiver!
