Sincronizar & Backup entre Aparelhos

Como funciona a sincronização?

Como o GitHub Pages é uma hospedagem estática e gratuita, ele não possui banco de dados em nuvem central. Para você transferir suas partituras do PC para um Tablet ou Celular, use as ferramentas abaixo de forma simples e rápida!

Passo 1: Exportar do Computador

Crie um arquivo de backup completo contendo todas as suas partituras cadastradas, suas anotações e pastas organizadas.

Passo 2: Importar no Celular/Tablet

Abra este mesmo link no celular ou tablet, clique aqui no painel e selecione o arquivo gerado para carregar todas as partituras em segundos.
